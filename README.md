A .NET console application for calculating FIFO (First-In-First-Out) profit/loss on security trades.

* FIFO Calculation - Accurate First-In-First-Out profit/loss calculation per security
* CSV Import - Load trade data from CSV files with automatic validation
* Error Handling - Comprehensive error reporting with line-by-line validation
* Detailed Reporting - Generates formatted reports with profit/loss by security
* Fee Accounting - Properly accounts for trading fees in profit/loss calculations

Prerequisites:
.NET 6.0 or later

Running the Application:
1. git clone https://github.com/Andzelika002/FIFO-Trade-Calculator.git
2. cd FIFO-Trade-Calculator

3. dotnet run -- path/to/your/trades.csv
or 
4. dotnet run
